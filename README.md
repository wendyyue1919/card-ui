# ui 组件库

### 快速开始

#### 1、安装组件库

```bash
npm i test-card-ui-yj
```

#### 2、引用组件库
```javascript
//全部引入
import 'test-card-ui-yj2/dist/css/index.css';
import MUI from 'test-card-ui-yj2';
Vue.use(MUI);

//按需引用
import 'test-card-ui-yj2/dist/css/demo.css';
import { Demo } from 'test-card-ui-yj2';
Vue.use(Demo);
```
