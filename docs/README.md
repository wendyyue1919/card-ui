# 快速开始

#### 安装组件库

```bash
npm i card-ui-yj
```

#### 引用组件库
>在 main.js 中引用组件库

```javascript
//全部引入
import 'test-card-ui-yj/dist/css/index.css';
import MUI from 'test-card-ui-yj';
Vue.use(MUI);

//按需引入
import 'test-card-ui-yj/dist/css/demo.css';
import {Demo} from 'test-card-ui-yj';
Vue.use(Demo);
```
